## What I do

- capture source metadata and provenance
- summarize why the source matters
- normalize a note into the resource schema
- preserve source links for web, local-file, and Zotero materials

## When to use me

Use this when a URL or external artifact should become a reusable vault source note.

## Constraints

- bare links are insufficient
- keep the original source identity and capture date
