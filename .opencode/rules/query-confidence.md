# Query Confidence Rules

When answering knowledge questions from this vault:

1. Prefer `wiki/` for stable answers.
2. Fall back to `resources/` for evidence and provenance.
3. Use `brainstorm/` only as tentative synthesis.
4. Use `my-work/` for current intent, context, and active priorities.

## Answer style

- State the confidence level when the answer is not fully grounded.
- Separate confirmed facts from working hypotheses.
- Cite the folder or note source that supports the answer when practical.
- If evidence conflicts, say so plainly instead of collapsing the conflict.
